# OTE-Zuo.github.io

这是一个用于 GitHub Pages 的课题组网站初始版本。

## 使用方法

1. 打开 GitHub 仓库：`https://github.com/OTE-Zuo/OTE-Zuo.github.io`
2. 上传本文件夹中的全部文件和文件夹
3. 进入 `Settings > Pages`
4. 设置为：
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /root
5. 访问：`https://OTE-Zuo.github.io`

## 页面结构

- `index.html`: 首页
- `research.html`: 研究方向
- `publications.html`: 成果及论文
- `people.html`: 组员介绍
- `join-us.html`: 加入我们
- `news.html`: 课题组新闻
- `assets/css/style.css`: 网站样式
- `assets/images/`: 图片资源

## 需要你替换的内容

- `your-email@fudan.edu.cn`
- `group-photo.jpg`
- 真实论文列表
- 真实组员信息
- 真实课题组新闻
