# Beautified OTE-Zuo.github.io

这是重新美化后的 GitHub Pages 网站版本。

## 上传方式

1. 解压本文件。
2. 打开仓库：`https://github.com/OTE-Zuo/OTE-Zuo.github.io`
3. 点击 `Add file > Upload files`
4. 上传所有文件和文件夹，覆盖原文件。
5. 选择 `Commit directly to the main branch`
6. 等待 GitHub Pages 自动部署。
7. 打开 `https://ote-zuo.github.io/`

## 主要改进

- 统一的学院官网风格
- 更稳的桌面端和手机端响应式布局
- 更清晰的导航栏
- 更规整的研究方向卡片
- 更干净的首页和研究方向总图
- 避免卡片和图片宽度导致排版混乱
